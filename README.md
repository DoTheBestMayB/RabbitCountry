## 2021.08.30
수정된 코드에서 닉네임에 기반한 유저 차단 추가

하지만 chrome.storage에서 차단할 닉네임이 담긴 list를 꺼내오고 비교할때 시간이 소요되어 미세한 깜빡임이 생김
